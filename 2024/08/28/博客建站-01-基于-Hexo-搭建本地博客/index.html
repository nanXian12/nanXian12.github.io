创建项目

在任意盘中创建一个空目录，作为博客目录。以下操作在博客目录下打开命令行窗口执行。

 

安装 git

官网下载：https://git-scm.com/download/win

 
查看 git 版本

 

安装 node

建议使用 Node.js 14.0 及以上版本，Hexo 最新版 7.0+ 最低支持 Node.js 14.0.0

官网下载：https://nodejs.org/zh-cn/download/

 
查看 node 版本

 

安装 hexo

 
博客目录下会生成 source、themes、_config.yml 等文件夹和文件。其中，source 是存放用户资源的地方，创建的文章会存放在 source 目录下的 _posts 文件夹中；themes 是主题文件夹，存放下载的主题；_config.yml 是博客项目配置文件。
