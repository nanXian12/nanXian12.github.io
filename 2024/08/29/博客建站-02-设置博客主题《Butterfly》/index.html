安装 Butterfly

 

安装渲染器

 

更换 Hexo 主题

打开博客目录下的 _config.yml 文件，将 theme 修改为 butterfly

为了减少升级主题后带来的不便，需要在博客目录下创建 _config.butterfly.yml 文件，复制 themes 目录下主题文件夹中 _config.yml 的内容到 _config.butterfly.yml 中

启动浏览

 

基础配置

标签页

打开博客项目下 source/tags 的 index.md 文件，添加 type: "tags"

示例：

参数 解释
type 【必须】页面类型，必须为 tags
orderby 【可选】排序方式 ：random/name/length
order 【可选】排序次序： 1, asc for ascending; -1, desc for descending

 

分类页

打开博客项目下 source/categories 的 index.md 文件，添加 type: "categories"

示例：

 

404 页

在 _config.butterfly.yml 文件中将 error_404 的 enable 属性改为 true

浏览器访问地址 http://localhost:4000/404.html 即可浏览效果

 

语言和网站资料

默认语言是 en，主题支持三种语言：default(en)、 zh-CN(简体中文)、zh-TW (繁體中文)

 

导航栏设置

网站图标

在博客项目下的 source 文件夹中新建文件夹 img 并存放图片 logo.png 和 favicon.png，设置配置文件：

参数 解释
logo 网站的 logo，支持图片，直接填入图片链接
display_title 是否显示网站标题，填写 true 或者 false
fixed 是否固定状态栏，填写 true 或者 false

 

菜单/目录

格式为：菜单名: /路由名/ || 图标名（如果不希望显示图标，图标名可以不写）

图标推荐：https://fontawesome.com/search?s=solid&f=classic&o=r

 

代码高亮主题

Butterfly 主题支持 6 种代码高亮样式：darker、pale night、light、ocean、mac、mac light

 

社交图标

格式：图标名: url || 描述性文字 || color
图标推荐：Find Icons with the Perfect Look & Feel | Font Awesome

顶部图

以上所有的属性值可配置以下值：

配置的值 效果
留空 显示默认的 top_img（如有），否则显示默认的颜色（文章页 top_img 留空的话，会显示 cover 的值）
img 链接 图片的链接，显示所配置的图片
颜色 ( HEX 值 - #0000FF
RGB 值 - rgb(0,0,255)
颜色单词 - orange
渐变色 - linear-gradient( 135deg, #E2B0FF 10%, #9F44D3 100%)） 对应的颜色
transparent 透明
false 不显示顶部图

并不推荐为每个标签和每个分类都配置不同的顶部图，因为配置太多会拖慢生成速度。
