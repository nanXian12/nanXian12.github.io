创建项目

在任意盘中创建一个空目录，作为博客目录。以下操作在博客目录下打开命令行窗口执行

安装 git

官网下载：https://git-scm.com/download/win

查看 git 版本
git --version

安装 node

建议使用 Node.js 14.0 及以上版本，Hexo 最新版 7.0+ 最低支持 Node.js 14.0.0
官网下载：https://nodejs.org/zh-cn/download/

（后续补充 node 安装步骤……）

查看 node 版本
node -v

安装 hexo

npm install hexo-cli -g

博客目录下会生成 source、themes、_config.yml 等文件夹和文件。其中，source 是存放用户资源的地方，创建的文章会存放在 source 目录下的 _posts 文件夹中；themes 是主题文件夹，存放下载的主题；_config.yml 是博客项目配置文件。

查看 hexo 版本
hexo -v

初始化 hexo

hexo i

启动本地服务

hexo s

浏览器中输入 http://localhost:4000/ 就可以浏览效果了

创建文章

hexo new "post name"

前往文章存放的路径中打开文件，编辑后保存

（后续补充 Typora 安装步骤……）

输入 hexo s 启动服务器预览效果

Hexo 命令

hexo clean 删除生成的静态文件和缓存
hexo config 获取或设置配置
hexo deploy 部署远程网站
hexo generate 生成静态文件
hexo help 获取有关命令的帮助
hexo init 初始化 Hexo 博客项目
hexo list 列出站点的信息
hexo migrate 将网站从其他系统迁移到 Hexo
hexo new "post name" 创建新文章
hexo publish 将草稿文章从 _drafts 移动到 _posts 文件夹
hexo render 使用渲染器插件渲染文件
hexo server 启动服务器
hexo version 显示版本信息
