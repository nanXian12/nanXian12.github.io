创建 Github 远程仓库

创建一个公开仓库，仓库名为用户名.github.io

 
创建完成之后进入仓库，打开设置

 
点击左侧菜单的“Pages”页面，将“Build and deployment”下的“Source”设置为 Deploy from a branch；然后在“Branch（分支）”下使用分支下拉菜单并选择发布源。

 
这样之后，在浏览器输入 https://用户名.github.io/ 就可以访问网站了。需要注意的是，将更改推送到 GitHub 后，最多可能需要 10 分钟才能发布对站点的更改。
